# 6.nestjs 常用命令

## 帮助

```bash
nest --help
```

## 新建项目

```bash
nest new project-name
```

## 启动项目

```bash
nest start
```

## 启动项目并监听文件变化

```bash
nest start --watch
```

## 启动项目并监听文件变化，并热更新

```bash
nest start --watch --webpack
```
