# where语句

## where过滤数据
```
SELECT * FROM 表名
WHERE 列名 = *;

SELECT * FROM 表名
WHERE 列名 > *;

```
## 一些操作符号
通过是用在where语句中的
### 常规操作符号
```sql
> 大于
< 小于
>= 大于等于
<= 小于等于
= 等于
<>(!=) 不等于
```
### 关键字操作符
```sql
IS NULL 为空运算
IS NOTNULL 不为空运算
LEAST 最小值运算符
GREATEST 最大值运算符
BETWEEN AND 两值之间运算符
IN 属于操作符
NOT IN不属于操作符
LIKE 模糊匹配
REGEXP 正则表达式运算符



```
